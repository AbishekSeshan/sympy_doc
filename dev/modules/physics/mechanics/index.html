Physics/Mechanics Module¶

Authors:Gilbert Gede, Luke Peterson

Abstract

In this documentation many components of the physics/mechanics module will be discussed. mechanics has been written to allow for creation of symbolic equations of motion for complicated multibody systems.

Mechanics¶

In physics, mechanics describes conditions of rest or motion; statics or dynamics. First, an idealized representation of a system is described. Next, we use physical laws to generate equations that define the system’s behaviors. Then, we solve these equations, in multiple possible ways. Finally, we extract information from these equations and solutions. Mechanics is currently set up to create equations of motion, for dynamics. It is also limited to rigid bodies and particles.
